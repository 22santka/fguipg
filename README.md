# fguipg
fGUIpg is a program that adds a Tkinter based Python GUI to the otherwise cmd line based FFmpeg.

CREDITS:

- FFmpeg, Fabrice Bellard; Bobby Bingham (libavfilter), & the FFmpeg team, https://ffmpeg.org/
